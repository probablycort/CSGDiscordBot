# CSGDiscordBot (CortSiriGoogle Discord Bot)
CSGDiscordBot is a Discord bot that has some basic auto-responding commands, different auto-changing statuses based on your battery percentage, a Discord bot command that checks your PC's battery (if it has one), and more. It's not really that life-changing anyway.

## A little heads up here...
I don't even know how to set this up by building it for yourself, since I was new to coding and GitHub. I would just copy and paste the code and that's it. Apologies that I couldn't make it easier for you.
This bot is currently used in the CortSiriGoogle Studios Discord server, but you can self-host it yourself.
Some of the codes were based on a Discord bot tutorial video by Indently on YouTube and a little bit of ChatGPT help as I am still a beginner at coding and stuff (I'm so sorry.).
