# CSGDiscordBot (CortSiriGoogle Discord Bot)
CortSiriGoogle Discord Bot is a Discord bot that has some basic auto-responding commands, different auto-changing statuses based on your battery percentage, a Discord bot command that checks your PC's battery (if it has one), and more. It's not really that life-changing anyways.

Some of the codes were based on a Discord bot tutorial video by Indently on YouTube and a little bit of ChatGPT help as I am still a beginner at coding and stuff (I'm so sorry.).
